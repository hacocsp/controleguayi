🔗 URL do Apps Script

Cole abaixo a URL do Apps Script implantado. O portal carrega e salva dados diretamente no Google Sheets, sem armazenamento local. A URL padrão já está configurada — substitua somente se necessário.

📋 Passo a passo para reimplantar o Apps Script

  1. Acesse script.google.com → abra o projeto existente
  2. Substitua o código pelo novo poupanca_script_final.gs
  3. Salve (Ctrl+S)
  4. Clique em Implantar → Gerenciar implantações → ✏️ editar → Nova versão → Implantar
  5. Cole a URL acima e clique em Salvar & Testar
