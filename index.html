⚙️ Configuração · Conexão com Google Sheets

🔗 URL do Apps Script

Cole abaixo a URL gerada ao implantar o arquivo poupanca_script_final.gs. O portal carregará e salvará dados diretamente no Google Sheets, sem armazenamento local.

📋 Passo a passo

  1. Acesse script.google.com → Novo projeto
  2. Apague o código existente → cole o conteúdo do poupanca_script_final.gs → salve
  3. Clique em Implantar → Novo deployment → App da Web
  4. Executar como: Eu · Quem acessa: Qualquer pessoa
  5. Clique Implantar → autorize → copie a URL gerada
  6. Cole a URL acima e clique em Salvar & Testar
